Vim 编辑器学习笔记

编辑器的选择

学习新的编辑器

  1. 阅读教程：如课程内容和资源。
  2. 坚持使用：初期效率可能较低，但长远来看更高效。
  3. 查阅帮助：利用编辑器的帮助文档寻找更方便的操作方法。

常见编辑器

  • Visual Studio Code：当前最流行的代码编辑器。
  • Vim：流行的基于命令行的编辑器，有悠久的历史和强大的功能。

Vim 的设计哲学

  • 多模态编辑：不同模式下，键盘操作的含义不同。
  • 避免鼠标：提高效率，减少手指移动。

编辑模式

  1. 正常模式：移动光标并进行修改。
  2. 插入模式：插入文本。
  3. 替换模式：替换文本。
  4. 可视化模式：选中文本块（一般、行、块）。
  5. 命令模式：执行命令（如打开、保存文件等）。

基本操作

  • 模式切换：

    • 从其他模式返回正常模式：<ESC>
    • 插入模式：i
    • 替换模式：R
    • 可视化模式：v（一般）、V（行）、<C-v>（块）
    • 命令模式：:

  • 缓存、标签页、窗口：

    • 一个 Vim 会话包含多个标签页，每个标签页可以包含多个窗口。
    • 窗口显示缓存，可以在不同窗口中查看同一个文件的不同部分。

  • 命令行操作：

    • :q：退出（关闭窗口）
    • :w：保存
    • :wq：保存并退出
    • :e {文件名}：打开文件
    • :ls：显示打开的缓存
    • :help {标题}：打开帮助文档

Vim 接口作为编程语言

  • 移动命令：
    • hjkl：左、下、上、右
    • w、b、e：词移动
    • 0、^、$：行内移动
    • gg、G：文件头尾
    • :/{正则表达式}：搜索
    • n、N：导航匹配
  • 选择模式：
    • v：可视化模式
    • V：可视化行
    • <C-v>：可视化块
  • 编辑操作：
    • i：进入插入模式
    • O、o：在上/下插入行
    • d{移动命令}：删除
    • c{移动命令}：改变
    • x：删除字符
    • s：替换字符
  • 更多操作：
    • u：撤销
    • <C-r>：重做
    • y：复制（“yank”）
    • p：粘贴
  • 计数和修饰语：
    • 3w：向后移动三个词
    • 5j：向下移动5行
    • ci(：改变当前括号内的内容
    • da'：删除单引号字符串，包括周围的单引号

常用命令

删除操作（d 命令）

  • d{移动命令}：删除从光标到指定位置的文本。
    • dw：删除到下一个词的开头。
    • d$：删除到行尾。
    • d0：删除到行首。
    • dd：删除整行。
    • d + a 或 i + {、[、"：删除括号内、方括号内或双引号内的文本。

改变操作（c 命令）

  • c{移动命令}：删除从光标到指定位置的文本，并进入插入模式。
    • cw：改变到下一个词的开头。
    • c$：改变到行尾。
    • cc：改变整行。
    • c + a 或 i + {、[、"：改变括号内、方括号内或双引号内的文本。

复制操作（y 命令）

  • y{移动命令}：复制从光标到指定位置的文本。
    • yw：复制到下一个词的开头。
    • yy：复制整行。
    • y + a 或 i + {、[、"：复制括号内、方括号内或双引号内的文本。

粘贴操作（p 和 P 命令）

  • p：在光标位置之后粘贴文本。
  • P：在光标位置之前粘贴文本。

撤销和重做（u 和 <C-r> 命令）

  • u：撤销上一个操作。
  • <C-r>：重做上一个撤销的操作。

查找和替换（/ 和 :s 命令）

  • /：搜索文本。比如 /pattern 查找 pattern。
  • :s/old/new/g：在当前行中替换所有 old 为 new。
  • :%s/old/new/g：在整个文件中替换所有 old 为 new。

移动和跳转（m 和 : 命令）

  • m{标记}：设置标记（例如 ma）。
  • '{标记}：跳转到设置的标记位置（例如 'a）。
  • :：进入命令模式，可以执行保存、退出等命令（如 :w 保存文件，:q 退出 Vim）。

括号和引号操作

跳转到配对括号：

  • %：跳转到配对的括号或引号。

替换

  • **pattern**：要匹配的模式。
  • **replacement**：用于替换匹配模式的文本。
  • **flags**：替换的选项，如全局替换等。

flags标志

  • **g**：全局替换（在每行中替换所有匹配项）。
  • **c**：每次替换前提示确认。
  • **i**：忽略大小写。

替换的范围

  • **:%s**：在整个文件中进行替换。
  • **:s**：在当前行中进行替换。
  • **:10,20s**：在第 10 行到第 20 行之间进行替换。

在实际运用中有两种替换最为常见，一是全局替换，二是先shift+v切换到行选中模式，此时进行替换命令会自动输入选中行。

多窗口

分割窗口

在 Vim 中，管理多个窗口可以帮助你更高效地查看和编辑文件。

  • 水平分割窗口：: sp [文件名]，将当前窗口水平分割，并在新的窗口中打开指定的文件。如果没有指定文件，则会在当前缓冲区中打开新窗口。

  • 垂直分割窗口：: vsp [文件名]，同上

窗口切换

  • **Ctrl-w h**：切换到左侧窗口
  • **Ctrl-w j**：切换到下侧窗口
  • **Ctrl-w k**：切换到上侧窗口
  • **Ctrl-w l**：切换到右侧窗口

窗口关闭

  • :q ： 关闭当前窗口
  • :qa ： 关闭所有窗口并且退出vim

小贴士

  • 返回正常模式：无论你在什么模式下，按 <ESC> 键都可以返回正常模式。
  • 重复命令：可以使用数字来重复命令，如 3dd 删除三行。
